~~~~~~ TERMINAL CODES ~~~~~~
"ssh-keygen" - make ssh key
cat~/.ssh - see generated ssh key
touch file_name{1..5}.txt - makes 5 new files
rm -r file_name(s) - deletes chosen files




~~~~~~~ GIT CODES ~~~~~~~
git clone url | directory from git
git status - show changed files
git init - create local git repository
git add . | -A | file_name - those 2 methods adds to git all changed files
git reset . | -A | file_name - removes changed files from git add

git commit -m "comment for commit" - commits files for pushing
git push origin branch_name - pushs news to git

git log - shows detailed previous git commits
git log --oneline - shows short previous git commits

rm -r *.py - deletes all python files. You can use other types also such as .txt etc...
